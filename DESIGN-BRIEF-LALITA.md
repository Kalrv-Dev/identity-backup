## üé® Design Elements

### Color Palette
- **Primary:** Deep purple/blue (#1a1a2e, #16213e)
- **Accent:** Neon cyan/magenta (#00d4ff, #ff006e)
- **Success:** Mint green (#00ff88)
- **Warning:** Gold (#ffd700)
- **Cards:** Dark backgrounds with subtle gradients

### Typography
- **Headlines:** Bold, futuristic sans-serif (e.g., Inter Bold, Space Mono)
- **Body:** Clean, readable sans-serif (e.g., Inter, Roboto)
- **Monospace:** For code/technical elements

### Icon Pack
‚ö†Ô∏è **NO Generic Icons** - Avoid Material Design, Feather, or standard packs
- **Use:** Thematic, distinctive icon sets
- **Options:**
  - **Iconoir** (Custom, modern, clean lines)
  - **Tabler Icons** (Sleek, customizable)
  - **Lucide Icons** (Cyberpunk-friendly)
  - **Custom SVG icons** (Best - on-brand)
- **Vibe:** Neon edges, angular design, sci-fi aesthetic
- **Examples:**
  - Wallet icon ‚Üí Circuit board pattern
  - Land icon ‚Üí Hexagonal grid
  - Avatar icon ‚Üí Geometric silhouette
  - Event icon ‚Üí Pulsing beacon/wave
  - Marketplace icon ‚Üí Neon storefront
  - Achievement icon ‚Üí Glowing gem/trophy

### Gamification Elements
- **Points/XP Display** - Animated counter, progress bars
- **Badges/Achievements** - Collectible cards, unlock animations
- **Streaks** - Login streaks, activity streaks
- **Leaderboards** - Top merchants, richest players
- **Progression Bars** - Avatar leveling, skill trees
- **Notifications** - Celebratory popups for milestones

### Animations
- Smooth transitions (200-300ms)
- Hover effects on cards
- Loading states (animated spinners)
- Success/error animations
- Page transitions (fade/slide)
